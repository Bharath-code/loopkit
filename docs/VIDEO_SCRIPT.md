**Word Count:** ~450 words  
**Tone:** Energetic, relatable, developer-focused

---

## [0:00-0:15] HOOK

**Visual:** Split screen. Left side: Developer staring at blank terminal, frustrated. Right side: Calendar showing weeks passing with "0 shipped."

**Audio (upbeat tech background music fades in):**

"You're a solo technical founder. You can build anything. But shipping every week? That's the hard part."

**Visual:** Quick montage: Notion tabs open, Trello board with stale tasks, Twitter draft box with blinking cursor.

"You've got the skills. You've got the ideas. But somewhere between 'build' and 'ship,' the loop breaks."

---

## [0:15-0:45] THE PROBLEM

**Visual:** Animated diagram showing scattered tools → confused founder → abandoned project.

"The problem? The weekly shipping loop was designed for teams. Every tool handles one phase. None connects them from your terminal."

**Visual:** Terminal screen with text: "Define → Develop → Deliver → Feedback → Iterate" (each phase grayed out).

"You start strong. Define the idea. Build the MVP. But then... writing launch copy takes hours. Feedback lives in scattered DMs. Next week's priorities? You're guessing."

**Visual:** Founder at desk, head in hands. Clock shows 2 AM.

"Sound familiar? You're not alone. Solo founders are strong builders but weak weekly shippers."

---

## [0:45-1:15] THE SOLUTION

**Visual:** Clean terminal animation. `npx loopkit init` appears, then expands into the LoopKit logo.

"Meet LoopKit. The CLI for solo technical founders shipping weekly."

**Visual:** Five colored blocks appear in sequence: DEFINE (blue), DEVELOP (green), DELIVER (yellow), LEARN (orange), ITERATE (purple).

"Five commands. One weekly loop. Define the bet, build against it, ship, collect feedback, and decide the next move—all from your terminal."

**Visual:** Screen recording of actual LoopKit usage:
- `loopkit init` → AI scoring 8/10
- `loopkit track` → tasks auto-closing via git
- `loopkit ship` → AI-generated HN post
- `loopkit pulse` → feedback clustering
- `loopkit loop` → Sunday synthesis

"No more scattered tools. No more blank tweet anxiety. No more guessing what to build next."

## [1:15-1:45] KEY FEATURES

**Visual:** Fast-paced feature highlights with icons.

**Feature 1 - AI Scoring:**
"Start with `loopkit init`. Five questions. AI scores your idea on ICP clarity, problem strength, and MVP focus. Get objective feedback before writing a single line of code."

**Feature 2 - Git-Native Tasks:**
"Track tasks in markdown. Reference `[#3]` in your commit message—task auto-closes. Shipping score shows your progress in real-time."

**Feature 3 - Launch Copy in Seconds:**
"Friday shipping? `loopkit ship` generates your HN post, Twitter thread, and Indie Hackers update in 30 seconds. Edit, regenerate, or skip. You're in control."

**Feature 4 - Feedback That Comes to You:**
"Collect feedback async via hosted forms or inline. AI clusters responses into 'Fix now,' 'Validate later,' or 'Noise.' Data beats intuition."

**Feature 5 - Sunday Ritual:**
"Every Sunday, `loopkit loop` synthesizes your week—tasks, ships, feedback, streak. AI recommends one high-leverage focus for next week. One decision. One post. Week done."

## Production Notes

**Visual Style:**
- Clean, modern terminal aesthetic
- Violet/cyan color scheme (matches LoopKit brand)
- Smooth animations between sections
- Screen recordings of actual CLI usage (not mockups)

**Audio:**
- Upbeat, tech-focused background music
- Professional voiceover (energetic but not hype-y)
- Sound effects for key moments (typing sounds, success chimes)

**Text Overlays:**
- Keep minimal—let visuals carry the message
- Use for command examples and key stats only
- Terminal font: Fira Code or similar

**Distribution:**
- YouTube (primary)
- Twitter/X (shortened 60s cut)
- Product Hunt launch video
- Website hero video
